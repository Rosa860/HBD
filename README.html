因為有一個人，光是出現在我生活裡，就已經很特別了。

平常沒說，但你其實是我最近最常想到的人。

你真的蠻厲害的，而且好像比我想像中還更讓人想靠近一點。

然後有空的話，也可以…多一點時間留給我（笑）
